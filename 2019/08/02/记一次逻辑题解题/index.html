一道蛮有趣的逻辑题


Friday, August 2nd 2019, 5:53 pm

这道题好像去年就见过了啊喂~


今天突然看到这份号称“2018年刑侦推理试题”的题，体内的推理之魂（太中二了吧）蠢蠢欲动。
大致看了下题，第一反应便是程序暴力解题哈哈。不过还是要考一考自己的逻辑思维的，这种题一般有个入口，找到这个入口就好办啦。
所以以下文章分享下我的解题思路以及程序解法。（还是很佩服出题人，听说是个社团出的题，好厉害！）

解题思路

入口是解题的第一步，这里我选择了第五题作为入口。

1、假设第5题选A

根据选项内容可知第8题也选A,且我们看下第2题，是和第5题有联系的，所以第2题选C
然后接下来看第8题的描述，我们选择了A说明BCD是错的，从而说明我们已知的第2和第5题的答案是和第1题是相邻的
所以第1题只能选B，而由第8题也知道了第7题的答案是与第1题不相邻，所以第7题选D。
看回第4题，由上面可知AB排除，而第七题指出十个答案里最少的是D，要想十个里有唯一一个最少个数，只能是一个或零个，所以其他选项不能是D了。第4题选C
1、9题答案相同，故第9题选B。根据第4题的解题思路，第6题选C，故第3题和第10题都选A。所以答案是BCACACDABA
一次假设就得到答案，看来第5题作为切入点比较好。看看还有没其他可能。

2、假设第5题选B

由第5可知第2选D，第4选B。再由第4得第7选D，这时候D就有两个了，所以是肯定不符合10个答案唯一一个最小的这个条件。

3、假设第5题选C

同理可得第2选A，第9选C，看第9的描述，由于第2和第5是不同的答案，所以只能第1和第6的答案相同才满足了。
看回第4题，假设选A，则1、5相同，由于5、9相同，所以1、9也相同，但选项C就是1、9相同，所以AC之间能互推，所以排除AC。
假如选B，2、7相同，明显违背了最少那个答案只有1个的定理，所以第4选D。这样可以得出1、6、10答案相同。所以第6选B,可得1、10、8都选B。
这个时候得到的结果已经和第8题的描述冲突了，所以这个假设不对。

4、假设第5题选D

第2选B，第7选D此时都有两个D了，所以D不可能是最小。

按这个思路好像只有一种答案BCACACDABA。让我们写个程序验证一下。

程序解题

目前只想到生成所有的答案去跑一遍所有的测试条件，看看剩余的答案是哪个。
第一步是生成所有答案，这个方法感觉挺多的，一开始想10个for，但是那样就太不美观了。
最后采取了从1道题所有答案可能慢慢组合到10道题。感觉效率不是很高，以后想到更快的算法再来完善。

std::string * makeAnswer(int num)
{
    using std::string;
    string *temp1 = new string[4]{"ya","yb","yc","yd"};
    string *temp2;
    for (int i = 2; i <= num; i++)
    {
        temp2 = new string[int(powl(4,i))];
        int count = 0;
        for (int l = 0; l < powl(4,i-1); l++)
        {
            temp2[count++] = temp1[l] +"a";
            temp2[count++] = temp1[l] +"b";
            temp2[count++] = temp1[l] +"c";
            temp2[count++] = temp1[l] +"d";
        }
        delete [] temp1;
        temp1 = temp2;      
    }
    return temp1;    
}

生成所有选项可能后，就是去验证是否符合所有条件了，这个比较简单(写的繁琐了点)

int main()
{
    std::string * answ = makeAnswer(10);
    for (int i = 0; i < pow(4,10); i++)
    {
        if (answ[i][0] == 'y')
        {
            //第二题  第5题的答案 A(C)  B(D)  C(A)   D(B)
            if (!(answ[i][2] == 'a'&&answ[i][5]=='c' || answ[i][2] == 'b'&&answ[i][5]=='d' || answ[i][2] == 'c'&&answ[i][5]=='a' || answ[i][2] == 'd'&&answ[i][5]=='b'))
            {
                answ[i][0] = 'n';
                continue;
            }

            //第三题  哪一题的答案与其他三项不同   A(3)  B(6)  C(2)   D(4)
            switch (answ[i][3])
            {
            case 'a':
                if (answ[i][2] == answ[i][3] || answ[i][4] == answ[i][3] || answ[i][6] == answ[i][3])
                    answ[i][0] = 'n';
                break;
            case 'b':
                if (answ[i][2] == answ[i][6] || answ[i][4] == answ[i][6] || answ[i][3] == answ[i][6])
                    answ[i][0] = 'n';
                break;
            case 'c':
                if (answ[i][6] == answ[i][2] || answ[i][4] == answ[i][2] || answ[i][3] == answ[i][2])
                    answ[i][0] = 'n';
                break;
            case 'd':
                if (answ[i][2] == answ[i][4] || answ[i][6] == answ[i][4] || answ[i][3] == answ[i][4])
                    answ[i][0] = 'n';
                break;
            }
            if (answ[i][0] == 'n')
                continue;

            //第四题    哪两道题答案相同    A(1、5)  B(2、7)  C(1、9)   D(6、10)
            switch (answ[i][4])
            {
            case 'a':
                if (answ[i][1] != answ[i][5])
                    answ[i][0] = 'n';
                break;
            case 'b':
                if (answ[i][2] != answ[i][7])
                    answ[i][0] = 'n';
                break;
            case 'c':
                if (answ[i][1] != answ[i][9])
                    answ[i][0] = 'n';
                break;
            case 'd':
                if (answ[i][6] != answ[i][10])
                    answ[i][0] = 'n';
                break;
            }
            if (answ[i][0] == 'n')
                continue;

            //第五题      哪道题答案与本题相同    A(8)  B(4)  C(9)   D(7)
            switch (answ[i][5])
            {
            case 'a':
                if (answ[i][8] != answ[i][5])
                    answ[i][0] = 'n';
                break;
            case 'b':
                if (answ[i][4] != answ[i][5])
                    answ[i][0] = 'n';
                break;
            case 'c':
                if (answ[i][9] != answ[i][5])
                    answ[i][0] = 'n';
                break;
            case 'd':
                if (answ[i][7] != answ[i][5])
                    answ[i][0] = 'n';
                break;
            }
            if (answ[i][0] == 'n')
                continue;

            //第六题      哪两题答案与第8题相同     A(2、4)  B(1、6)  C(3、10)   D(5、9)
            switch (answ[i][6])
            {
            case 'a':
                if (answ[i][8] != answ[i][2] || answ[i][8] != answ[i][4])
                    answ[i][0] = 'n';
                break;
            case 'b':
                if (answ[i][8] != answ[i][1] || answ[i][8] != answ[i][6])
                    answ[i][0] = 'n';
                break;
            case 'c':
                if (answ[i][8] != answ[i][3] || answ[i][8] != answ[i][10])
                    answ[i][0] = 'n';
                break;
            case 'd':
                if (answ[i][8] != answ[i][5] || answ[i][8] != answ[i][9])
                    answ[i][0] = 'n';
                break;
            }
            if (answ[i][0] == 'n')
                continue;

            //第七题    被选中最少的选项字母    A\B\C\D
            int answer_num[4]{0};
            enum abcd {a,b,c,d};
            for (int j = 1; j <= 10; j++)
            {
                switch (answ[i][j])
                {
                case 'a':
                    answer_num[0]++;
                    break;
                case 'b':
                    answer_num[1]++;
                    break;
                case 'c':
                    answer_num[2]++;
                    break;
                case 'd':
                    answer_num[3]++;
                    break;
                }
            }
            int min = answer_num[0],max = answer_num[0];
            abcd minC = a,maxC = a;
            for (int n = 1; n < 4; n++)
            {
                if(min > answer_num[n])
                {
                    min = answer_num[n];
                    minC = abcd(n);
                }
                if (max < answer_num[n])
                {
                    max = answer_num[n];
                    maxC = abcd(n);
                }   
            }

            switch (answ[i][7])
            {
            case 'a':
                if (minC != c)
                    answ[i][0] = 'n';
                break;
            case 'b':
                if (minC != b)
                    answ[i][0] = 'n';
                break;
            case 'c':
                if (minC != a)
                    answ[i][0] = 'n';
                break;
            case 'd':
                if (minC != d)
                    answ[i][0] = 'n';
                break;
            }
            if (answ[i][0] == 'n')
                continue;

            //第八题     下面哪题答案与第1题答案不相邻    A(7)  B(5)  C(2)   D(10)
            switch (answ[i][8])
            {
            case 'a':
                if (int(answ[i][7]) - int(answ[i][1]) <= 1 && int(answ[i][7]) - int(answ[i][1]) >= -1)
                    answ[i][0] = 'n';
                break;
            case 'b':
                if (int(answ[i][5]) - int(answ[i][1]) <= 1 && int(answ[i][5]) - int(answ[i][1]) >= -1)
                    answ[i][0] = 'n';
                break;
            case 'c':
                if (int(answ[i][2]) - int(answ[i][1]) <= 1 && int(answ[i][2]) - int(answ[i][1]) >= -1)
                    answ[i][0] = 'n';
                break;
            case 'd':
                if (int(answ[i][10]) - int(answ[i][1]) <= 1 && int(answ[i][10]) - int(answ[i][1]) >= -1)
                    answ[i][0] = 'n';
                break;
            }
            if (answ[i][0] == 'n')
                continue;

            //第九题     “第1第6答案相同”  “第x第5答案相同”  两个真假性相反    A(6)  B(10)  C(2)   D(9)
            switch (answ[i][9])
            {
            case 'a':
                if (answ[i][1] == answ[i][6] && answ[i][5] == answ[i][6] || answ[i][1] != answ[i][6] && answ[i][5] != answ[i][6])
                    answ[i][0] = 'n';
                break;
            case 'b':
                if (answ[i][1] == answ[i][6] && answ[i][5] == answ[i][10] || answ[i][1] != answ[i][6] && answ[i][5] != answ[i][10])
                    answ[i][0] = 'n';
                break;
            case 'c':
                if (answ[i][1] == answ[i][6] && answ[i][5] == answ[i][2] || answ[i][1] != answ[i][6] && answ[i][5] != answ[i][2])
                    answ[i][0] = 'n';
                break;
            case 'd':
                if (answ[i][1] == answ[i][6] && answ[i][5] == answ[i][9] || answ[i][1] != answ[i][6] && answ[i][5] != answ[i][9])
                    answ[i][0] = 'n';
                break;
            }
            if (answ[i][0] == 'n')
                continue;

            //第十题    abcd出现最多与出现最少之差   A(3)  B(2)  C(4)   D(1)
            switch (answ[i][10])
            {
            case 'a':
                if (max - min  != 3 )
                    answ[i][0] = 'n';
                break;
            case 'b':
                if (max - min  != 2 )
                    answ[i][0] = 'n';
                break;
            case 'c':
                if (max - min  != 4 )
                    answ[i][0] = 'n';
                break;
            case 'd':
                if (max - min  != 1 )
                    answ[i][0] = 'n';
                break;
            }
        }

    }

    for (int i = 0; i < pow(4,10); i++)
    {
        if (answ[i][0] == 'y')
        {
            std::cout << answ[i] << std::endl;
        }

    }

}
